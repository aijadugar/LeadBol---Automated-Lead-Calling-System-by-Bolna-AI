# Frontend – AI Lead Calling Dashboard

This folder contains the frontend application for the AI Lead Calling System.  
It provides a dashboard to view leads, call status, call summaries, and recordings generated by the AI voice agent.

The frontend communicates with the backend API to fetch lead data and call logs.

---

## Tech Stack

- React
- Vite
- TypeScript
- Axios (API calls)

---

## Features

- View list of leads
- Track call status (completed / failed / in-progress)
- View AI generated call summaries
- Listen to call recordings
- Download the updated leads

---

## Installation

Clone the repository and install dependencies.

```bash
cd frontend
npm install
````

---

## Run the Development Server

```bash
npm run dev
```

The app will start at:

```
http://localhost:5173
```

---

## Backend Connection

Make sure the backend server is running.

Example backend API:

```
http://localhost:8000
```

Update the API base URL in:

```
src/services/api.js
```

Example:

```javascript
const VITE_BASE_URL = "http://localhost:8000";
```
